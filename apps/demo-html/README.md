# StreamFlow Demo Website (Multi-Page)

## Open

Open `demo/index.html` in your browser.

## Pages

- `index.html` Main page and project overview
- `onboarding.html` OTP + AI doc extraction simulation
- `offer.html` Financial connection + risk scoring + partial acceptance
- `payout.html` Supplier payout simulation
- `repayment.html` Sales/fixed/hybrid repayment + fast-forward
- `investors.html` Investor fund pool simulation
- `dashboard.html` Live KPI and timeline summary

## Demo Characteristics

- Frontend only, no backend
- Uses `localStorage` to share state across pages
- Includes full idea from the PDF plus your 3 added features

## Reset

Use **Reset Demo State** on the main page.
